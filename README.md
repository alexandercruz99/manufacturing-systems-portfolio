## What I Do

I diagnose and fix application failures in production systems. My workflow:

1. **Reproduce** - Identify the failure from logs, error codes, or user reports
2. **Isolate** - Trace the issue to root cause (validation, deserialization, configuration)
3. **Fix** - Implement the solution with proper error handling
4. **Verify** - Test the fix with real requests and confirm resolution
5. **Document** - Update documentation and create evidence of the fix

**Skills:** Request validation, error handling, API debugging, log analysis, JSON deserialization, .NET/C#, REST APIs, Swagger/OpenAPI
